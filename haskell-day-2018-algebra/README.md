## Requirements

- pip
- haskell-stack

## How to make

The content :point_down:

```shell-session
$ make install-deps  # Only first
$ make
$ xdg-open ./index.html
```

The Haskell codes :point_down:

```shell-session
$ make code
$ ls FirstHalf.hs LastHalf.hs
FirstHalf.hs  LastHalf.hs
```
