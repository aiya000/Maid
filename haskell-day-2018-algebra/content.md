### 代数の素朴な定義 - マグマ

```haskell
class Magma a where
    (<>) :: a -> a -> a

instance Magma Int where
    (<>) = (+)

instance Magma [a] where
    (<>) = (++)
```

- - - - -

### 代数の素朴な定義 - マグマ

```haskell
instance Num a => Magma a where
    (<>) = (+)

newtype Product a = Product
    { unProduct :: a
    } derving (Show, Eq)

derving instance Num a => Num (Product a)
```
