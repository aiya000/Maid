### ちょっとした前提知識 - 集合

集合Int は
「**-1より下**,
-1, 0, 1,
**1より上**」
を含む

```
Int = {minBound, ..., -1, 0, 1, ..., maxBound}
```

- 「foo, **...**, bar」: 有限な省略
- HaskellでいうEnum, OrdかつBounded

- - - - -

### ちょっとした前提知識 - 集合

集合Integer（整数の集合）は  
「**-1より下**,
-1, 0, 1,
**1より上**」  
を含む

```
Integer = {..., -1, 0, 1, ...}
```

- 「**...**, foo」, 「bar, **...**」: 有限な省略
